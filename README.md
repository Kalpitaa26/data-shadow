# Data Shadow
Data Shadow is a privacy-focused browser extension that helps users understand how websites collect and use their data. The extension analyzes cookies, trackers, and potential privacy risks in real time, while also suggesting measures users can take to protect their online privacy.

## Features
- Detects website cookies and trackers
- Analyzes potential privacy and data safety risks
- Educates users about data collection practices
- Provides privacy awareness and protection tips
- Real-time browser monitoring

## Tech Stack
- HTML
- CSS
- JavaScript
- Browser Extension APIs

## Purpose
The goal of Data Shadow is to create awareness about online privacy and help users make safer browsing decisions.
